
# 什么是脏页
mysql脏页和干净页：  
当内存数据页和磁盘数据页上的内容不一致时，我们称这个内存页为 脏页；  
内存数据写入磁盘后（术语就是flush），内存页上的数据和磁盘页上的数据就一致了，我们称这个内存页为干净页。  
不管是 脏页 还是 干净页 ，他们都在内存中。  

内存里的数据写入磁盘的过程，术语就是flush。  
刷脏页可能导致MySQL的抖动。
